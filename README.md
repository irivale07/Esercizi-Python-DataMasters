# 🚀 Python for QA Automation Portfolio

Benvenuti nel mio repository dedicato all'automazione dei test! Questo spazio raccoglie gli esercizi, i progetti e le simulazioni di test sviluppati durante il mio percorso di formazione come *QA Automation Engineer*.

---

## 📂 Struttura del Repository

L'architettura è organizzata per moduli incrementali, seguendo le "best practice" del testing moderno:

* *01_Python_Basics/*: Fondamenta del linguaggio (variabili, cicli, strutture dati) applicate a scenari di test.
* *02_API_Automation/*: Test automatizzati per servizi REST utilizzando la libreria Requests.
* *03_UI_Automation/: Automazione del browser con Selenium/Playwright (implementando il *Page Object Model).
* *04_Projects/*: Casi studio reali e integrazioni End-to-End. (per il futuro)

---

## 🛠️ Tecnologie & Tool
- *Language:* Python 3.x
- *Testing Frameworks:* Pytest
- *Web Automation:* Selenium / Playwright
- *API Testing:* Requests, Postman
- *Version Control:* Git & GitHub

---

## 🎯 Obiettivi del Percorso
Il mio obiettivo è padroneggiare l'intero ciclo di vita del software testing, scrivendo codice pulito, manutenibile e scalabile per garantire la massima qualità del prodotto.
